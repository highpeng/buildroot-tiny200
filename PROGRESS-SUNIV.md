- MMC 0
- MMC 1
- SPI NOR Flash
- SPI NAND Flash

- Basic LCD
- GPIO Backlight
- SF Devices
- MTD Devices

- UMS USB Mass Storage Drive Emulator
- DFU Firmware Downloader

  - SPI NAND
- TFCard Firmware Downloader

  - SPI NAND

- Audio Codec: Playback, Mic-IN

- Display Engine: Frontend, Backend, TCON
- AW9523B GPIO Expender
